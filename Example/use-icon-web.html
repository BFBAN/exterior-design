Using Icons

Use svg+xml mode to introduce the tour tag icon, it will automatically decide to display according to the style, the following shows some references to the way:

1.Cite it on a web page

<div style="width: 100px"> <embed src="../Materials/web-all.svg" type="image/svg+xml" /> </div>

2.Reference to the tour tab icon

You need to pay attention to compatibility and backward compatibility for unsupported versions

<link rel="icon" href="../Materials/web-all.svg" type="image/svg+xml">
